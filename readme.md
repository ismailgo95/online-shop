## Cara Menggunakannya

1. Clone or Fork Repo ke Local PC
2. Composer Update / Composer Install 
3. php artisan key:generate
4. Buat File .Env atau Copy .env.example dan ubah Database
5. php artisan migrate
6. php artisan serve
